## How to run the app in VS code
- Step one
Create an emulator from Visual Studio Code, don't run emulator from Android Studio, it has the error of `Error connecting to the service protocol: failed to connect to http://127.0.0.1`
- Step two
In the project, run command `flutter run`

## How to run the app in debug mode in VS code
- Step one
Create an emulator from Visual Studio Code, don't run emulator from Android Studio, it has the error of `Error connecting to the service protocol: failed to connect to http://127.0.0.1`
- Step two
In the project, click on debug window, run it from there
